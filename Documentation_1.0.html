ZET-CSV



      ZET-CSV is the C++ library for csv format. ZET-CSV uses STL library. Interface CSV container similar to the interface std::vector.
      To install, use the Makefile (make -f Makefile, make -f Makefile install). By default, header file(zCSV .h) and lib file will be installed in /usr/local/include/ and /usr/local/lib/. You can use ZET- CSV (not as the library) with your code by rewriting the Makefile.
      ZET- CSV contains following csv extension: a different number of columns in the rows, literals \r, \n, \r\n indicate the end of the row.

The function returns true if successfully , false if unsuccessfully.


void write(std::string &ret, char delim=',', char quote='\"') const;

void normalize();

Sets a common size for all rows. The length of every row is equal to the maximum size of the rows in the container.

std::vector& operator[](size_t pos);
const std::vector& at(size_t pos) const;
std::vector& at(size_t pos);

Returns a reference to row are placed at pos.


const std::vector& front() const;
std::vector& front();

Returns a reference to the first row.


const std::vector& back() const;
std::vector& back();

Returns a reference to the last row.


bool empty() const;

Indicates when the object has any rows.


size_t size() const;

Returns the number of rows that the object has.


void clear();

Removes all rows from the object (which are destroyed), leaving the container with a size of 0.


bool erase(size_t pos);

Removes from the object a single row at pos.
This effectively reduces the container size by the one row, which are destroyed.


bool pop_back();

Removes the last row in the object, effectively reducing the container size by one.
This destroys the removed row.


size_t insert(size_t pos, const std::vector& row);
size_t insert(size_t pos, std::vector* row);

The object is extended by inserting the element row before the element at the specified pos, effectively increasing the container size by the one. If pos >= size the object will be added to back.
Returns the position of inserted row. At the first function the copy of the row will be created. At the second function the object row must be inserted.


size_t push_back(const std::vector& row);
size_t push_back(std::vector* row);

Adds a new row at the end of the object, after its current last element. The object is extended by adding the element row at the back, effectively increasing the container size by the one.
Returns the position of added row. At the first function the copy of the row will be created. At the second function the object row must be added.
